Movie application
"myFlix"
project description:

Web application provides the users with list of movies to choose from. Organized in few main categories with option to create
custom list. Information about movies, generes, actors and more will be accessable as well. Users are able to log-in with personal information
and update it any time.

API Endpoints

Request URL Description Method Query Parameters Request Body Response
Get All Movies /movies Returns a list of all movies to the user. GET None None JSON array of movie objects
Get movie by title /movies/:title Returns movie by title. GET title (URL parameter) None
Get movie by genre /genres/genres/:genreName Return movie by genre. GET name (URL parameter) None
Get movie by director's name. /title/:directorName Return movie by director's name. GET name (URL parameter) None
Register new user /users Allows a new user to register. POST None JSON object with user data (username, password, email, dateOfBirth) Confirmation message or error message
Update exsiting user's name /users/:id Allows a user to update their personal information. PUT username (URL parameter) Updated user data or confirmation message
Deregister exsiting user /users/:id Allows an existing user to deregister. DELETE username (URL parameter) None Confirmation message or error message
Add movie to favorite movies /users/:id/:movieTitle Add a movie to a user's favorite movies. POST username (URL parameter) JSON object with movie title Updated list of favorite movies or confirmation message
Remove movie from favorite movies /users/:id/:movieTitle Remove a movie from favorite movies. DELETE username (URL parameter), movieTitle (URL parameter) None Updated list of favorite movies or confirmation message
