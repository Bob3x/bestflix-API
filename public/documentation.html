Movie application
"myFlix"
project description:

Web application provides the users with list of movies to choose from. Organized in few main categories with option to create
custom list. Information about movies, generes, actors and more will be accessable as well. Users are able to log-in with personal information
and update it any time.

Business Logic URL HTTP Method Query Parameters Request body data format Response body data format
Return a list of ALL movies to the user /movies GET none none A JSON object with all movies
Return parameters (title, director, genre, year) about a single movie by title to the user /movies/:title GET title (URL parameter) none Return movie by title Example:
{
  title: 'The Godfather',
}
Return movie by genre (e.g. “Crime”) /genres/:genreName GET name (URL parameter) none Return movie by genre Example:
{
  genre: 'Crime',
}
Return movie by director /movies/title/:directorName GET name (URL parameter) none Return movie by director's name Example:
{
  directorName: 'Frank Darabont',
}
Return movie by year /movies/title/:directorName GET name (URL parameter) none Return movie by year Example:
{
  year: 'Frank Darabont',
}
Register a new user /users POST none Creating new user name. Example:
{
  new user: 'Janny Gold',
} 		A confirmation or an error message.
Allow users to update their username /users/:id PUT username (URL parameter) A JSON object holding updated data about the username. Example:
{
  username: 'Mary Black'
} 		A confirmation message and updated user data or an error message. Example:
{
  username: 'Mary Black',
}
Allow users to add a movie to their list of favorites /users/:username/favorites POST username (URL parameter) A JSON object holding data about a favorite movie. Example:
{
  title: 'Peter Jackson's King Kong'
} 		A confirmation or an error message.
Allow users to remove a movie of their list of favorites /users/:username/favorites/:movieTitle DELETE username (URL parameter), movieTitle (URL parameter) A JSON object holding data about a favorite movie to remove. Example:
{
  title: 'Peter Jackson's King Kong'
} 		A confirmation or an error message.
